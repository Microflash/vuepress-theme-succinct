# Installation

vuepress-theme-succinct uses prefers-color-scheme media query and CSS variables to generate dark and light themes.

# Compatibility

The support for prefers-color-scheme and CSS Variables may vary depending on your browser and operating system.

# Get Started

Add the dependency to your existing Vuepress project via Yarn or NPM.

yarn add -D vuepress-theme-succinct
npm install -D vuepress-theme-succinct

TIP

Refer to Getting Started Guide to create a Vuepress project.

Update .vuepress/config.js as follows.



 


// .vuepress > config.js
module.exports = {
  theme: 'vuepress-theme-succinct'
}

You can also use the shorthand.



 


// .vuepress > config.js
module.exports = {
  theme: 'succinct'
}
